# 📋 DX팀의 마지막 배포 — 구현 태스크 목록

## Task 1: HTML 기본 구조 + 타이틀 화면

### 설명
index.html 파일에 게임의 기본 골격과 타이틀 화면을 구현한다.

### 상세
- [ ] HTML 기본 구조 생성 (meta, viewport, charset)
- [ ] 전체 게임 화면을 담을 섹션들 생성 (title, character-select, room1, room2, room3, ending)
- [ ] 타이틀 화면 UI: 제목 "DX팀의 마지막 배포", 부제 "금요일 밤, 장애가 터졌다."
- [ ] 타이틀 배경: 다크 네이비 그라데이션 + 모니터 깜빡임 CSS 애니메이션
- [ ] [시작하기] 버튼
- [ ] 기본 CSS 리셋 + 공통 스타일 (컬러 팔레트, 폰트, 트랜지션)
- [ ] 화면 전환 함수 (showScreen) 기본 구현

### 완료 기준
- index.html을 브라우저에서 열면 타이틀 화면이 보인다
- 시작하기 버튼 클릭 시 캐릭터 선택 화면으로 전환된다

---

## Task 2: 캐릭터 선택 화면

### 설명
4명의 캐릭터를 선택할 수 있는 화면을 구현한다.

### 상세
- [ ] 캐릭터 4명 카드 UI (이미지 + 이름 + 직급 + 특성 + 스킬)
- [ ] 캐릭터 카드 hover 효과 (확대, 글로우)
- [ ] 캐릭터 클릭 시 선택 확인 모달 or 하이라이트
- [ ] 게임 상태 객체 생성 (선택 캐릭터, 현재 방, 클리어 시간, 힌트 사용 횟수)
- [ ] 캐릭터별 스킬 정보 저장
- [ ] 선택 완료 → 인트로 스토리 텍스트 표시 후 방1로 전환
- [ ] 이미지 로딩 실패 시 폴백 (이니셜 아바타)

### 완료 기준
- 4명의 캐릭터가 카드 형태로 표시된다
- 선택한 캐릭터 정보가 게임 상태에 저장된다
- 선택 후 방1로 자연스럽게 전환된다

## Task 3: 방 1 — 6 회의실 (팀원 매칭 퍼즐)

### 설명
첫 번째 방의 배경, 스토리, 퍼즐을 구현한다.

### 상세
- [ ] 방 배경: room1.jpg + 어둡게 오버레이 처리
- [ ] 입장 스토리 텍스트 타이핑 효과: "금요일 저녁 7시. 혼자 남아 야근 중..."
- [ ] 퍼즐 1: 팀원-특징 매칭 UI
  - 왼쪽: 팀원 이름 7개 (김하늘, 이성현, 이정호, 박예은, 여승혁, 문기준, 김재환)
  - 오른쪽: 특징 힌트 7개 (셔플된 상태)
  - 드래그앤드롭 or 클릭-클릭 방식으로 매칭
- [ ] 매칭 정답 체크 로직
- [ ] 정답 시: 4자리 비밀번호 힌트 공개
- [ ] 퍼즐 2: 비밀번호 입력 UI (4자리 숫자 입력)
- [ ] 오답 시 화면 흔들림 효과
- [ ] 정답 시 문 열림 애니메이션 + 장애 알림 메시지
- [ ] 캐릭터 스킬 적용 (이상현: 힌트 미리보기, 홍병우: 추가 힌트)
- [ ] 힌트 버튼 (사용 횟수 제한)

### 완료 기준
- 팀원-특징 매칭 퍼즐을 풀 수 있다
- 비밀번호를 입력하면 방2로 진행된다
- 오답 피드백이 정상 동작한다

## Task 4: 방 2 — 장교빌딩 서버실 (에러 로그 + 복구 퍼즐)

### 설명
두 번째 방의 배경, 스토리, 퍼즐을 구현한다.

### 상세
- [ ] 방 배경: room2.jpg + 빨간 경고등 펄스 CSS 효과
- [ ] 입장 스토리: "서버실에 도착했다. 모니터에 에러 로그가 폭포처럼..."
- [ ] 퍼즐 1: 더스틴 에러 로그 해독
  - 에러 로그 5줄 표시 (터미널 스타일 UI)
  - 핵심 키워드 3개를 올바른 순서로 클릭
  - 전세웅 캐릭터 스킬: 첫 키워드 자동 하이라이트
- [ ] 퍼즐 2: 서버 복구 시퀀스
  - 4개 명령어 카드를 올바른 순서로 배치
  - 드래그앤드롭 or 숫자 클릭 방식
  - 순서: restart → clear cache → rebuild → deploy
- [ ] 오답 시 팀원 힌트 메시지 표시 (전세중, 홍병우, 이상현)
- [ ] 타이머 표시 (긴박감 연출, 실패 조건은 없음)
- [ ] 박종택 캐릭터 스킬: 타이머 +30초
- [ ] 클리어 시 서버 복구 완료 연출 + 롯데백화점 이동 메시지

### 완료 기준
- 에러 로그에서 키워드를 순서대로 클릭할 수 있다
- 복구 명령어를 올바른 순서로 배치하면 클리어된다
- 팀원 힌트가 오답 시 순차적으로 표시된다

## Task 5: 방 3 — 롯데백화점 본점 (DX팀 퀴즈)

### 설명
세 번째 방의 배경, 스토리, 최종 퀴즈를 구현한다.

### 상세
- [ ] 방 배경: room3_ext.jpg → room3_lobby.jpg (외부→내부 전환 연출)
- [ ] 입장 스토리: "단서를 따라 롯데백화점 본점에 도착했다..."
- [ ] 퍼즐: DX팀 퀴즈 (7문제 풀에서 5문제 랜덤 출제)
  - 객관식 4지선다 UI
  - 3문제 이상 정답 시 통과
- [ ] 퀴즈 문제 데이터:
  1. 매년 나트랑 여행 가는 사람? → 여승혁
  2. 전세중 수석님 특기? → 간식 먹기
  3. 내몸혁명 콤비? → 홍병우 & 박종택
  4. 박예은 리더 강아지 이름? → 재인
  5. 올해 첫 합류한 사람? → 이정호
  6. 문기준 리더가 좋아하는 브랜드? → 나이키
  7. 김재환 책임이 매주 가는 곳? → 에버랜드
- [ ] 각 문제 정답/오답 즉시 피드백 (색상 + 아이콘)
- [ ] 점수 카운터 표시
- [ ] 통과 시 엔딩 전환 연출 (화면 밝아짐)
- [ ] 실패 시 재도전 버튼

### 완료 기준
- 5문제가 랜덤으로 출제된다
- 3문제 이상 맞추면 엔딩으로 진행된다
- 각 문제에 즉시 피드백이 있다

## Task 6: 엔딩 화면 (캐릭터별 분기 + 감동 연출)

### 설명
캐릭터별 개인화된 엔딩과 감동적인 마무리 연출을 구현한다.

### 상세
- [ ] 화면 서서히 밝아지는 CSS 애니메이션
- [ ] 공통 메시지 타이핑 효과: "혼자 고생했다고 생각했지? 우린 항상 같이야."
- [ ] 캐릭터별 개인 메시지 표시 (4가지 분기)
- [ ] 팀원 이름 13명 순차적 페이드인 (크레딧 롤)
- [ ] "— DX팀 일동" 최종 표시
- [ ] 클리어 시간 표시
- [ ] 별/빛 파티클 CSS 애니메이션 (엔딩 분위기)
- [ ] "다른 캐릭터로 다시 플레이" 버튼 → 캐릭터 선택 화면으로
- [ ] "처음부터 다시" 버튼 → 타이틀로

### 완료 기준
- 선택한 캐릭터에 따라 다른 엔딩 메시지가 표시된다
- 팀원 전체 이름이 크레딧처럼 나타난다
- 감동적인 연출 (타이핑, 페이드인, 파티클)이 동작한다
- 리플레이 가능하다

## Task 7: 반응형 디자인 + 최종 폴리싱

### 설명
모바일 대응, 전체 플로우 테스트, 최종 다듬기를 수행한다.

### 상세
- [ ] 모바일 반응형 CSS (768px 이하)
- [ ] 터치 이벤트 대응 (드래그앤드롭 → 탭 방식 폴백)
- [ ] 전체 게임 플로우 테스트 (시작 → 방1 → 방2 → 방3 → 엔딩)
- [ ] 이미지 로딩 최적화 (lazy loading)
- [ ] 접근성: 키보드 네비게이션, 포커스 관리
- [ ] 전환 애니메이션 부드러움 확인
- [ ] 크로스 브라우저 테스트 (Chrome, Edge, Safari)
- [ ] 파일 크기 확인 (이미지 제외 HTML 크기 적정한지)

### 완료 기준
- 모바일에서 플레이 가능하다
- 전체 플로우가 막힘 없이 동작한다
- 주요 브라우저에서 정상 작동한다
