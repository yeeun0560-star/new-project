## 4. 방 1: 6 회의실

### 배경
- 이미지: room1.jpg (실제 회의실 사진)
- 분위기: 불이 꺼진 회의실, 화면 살짝 어둡게 처리
- 스토리: "금요일 저녁 7시. 혼자 남아 야근 중이던 당신. 갑자기 정전이 됐다. 문이 잠겼다."

### 퍼즐 1: 팀원 특징 매칭
- 화이트보드에 팀원들의 특징 힌트가 적혀있음
- 올바른 팀원-특징 조합을 맞추면 4자리 비밀번호 획득

**힌트 목록 (팀원들이 남긴 메모):**
| 팀원 | 힌트 내용 |
|---|---|
| 김하늘 리더 | "97년생 소개팅 만렙 🎯" |
| 이성현 리더 | "잘생긴 남자를 좋아합니다 ✨" |
| 이정호 리더 | "올해 처음 합류! 아직 어리숙... 🐣" |
| 박예은 리더 | "팀 유일 유부녀. 강아지 재인이 엄마 🐶" |
| 여승혁 리더 | "매년 나트랑 가는 여행 마스터 ✈️" |
| 문기준 리더 | "나이키 신발 덕후 👟" |
| 김재환 책임 | "민재 아빠, 매주 에버랜드 단골 🎢" |

### 퍼즐 2: 비밀번호 입력
- 팀원 매칭을 통해 얻은 숫자 조합으로 문 잠금장치 해제
- 정답: DX팀 인원수(13) + 올해 연도 끝자리(26) → "1326" 또는 팀원 관련 숫자

### 클리어 시
- 문이 열리며 장애 알림 소리 효과
- "삐빕— [긴급] 서버 장애 발생. 장교빌딩 서버실로 이동하세요."

## 5. 방 2: 장교빌딩 서버실

### 배경
- 이미지: room2.jpg (실제 장교빌딩 사무실 사진)
- 분위기: 빨간 경고등 효과, 긴박한 분위기
- 스토리: "서버실에 도착했다. 모니터에 에러 로그가 폭포처럼 쏟아지고 있다. 시간이 없다."

### 퍼즐 1: 더스틴 에러 로그 해독
- 화면에 더스틴 관련 에러 메시지가 여러 줄 표시됨
- 에러 메시지 중 핵심 키워드를 올바른 순서로 클릭
- 전세웅 책임 캐릭터 선택 시: 자동으로 첫 번째 키워드 하이라이트

**에러 로그 예시:**
```
[ERROR] Dustin build failed: module 'core' not found
[WARN] Memory usage: 94% — recommend restart
[ERROR] Deploy pipeline timeout at stage 3
[INFO] Last commit by 전세웅: "fix: 이번엔 진짜 됨"
[ERROR] Connection refused: port 8080 already in use
```

### 퍼즐 2: 서버 복구 시퀀스
- 올바른 순서로 복구 명령어를 배치 (드래그앤드롭 or 클릭)
- 순서: restart → clear cache → rebuild → deploy
- 틀리면 에러 카운트 증가 + 팀원 힌트 메시지 표시

**팀원 힌트 메시지 (틀릴 때마다 하나씩 표시):**
- 전세중 수석: "간식 먹으면서 천천히 생각해봐 🍪"
- 홍병우 팀장: "포기하면 내몸혁명처럼 다시 시작해야 해 💪"
- 이상현 책임: "전화해서 물어봐~ 아 잠깐 내가 말해줄게... (스크롤 넘침)"

### 클리어 시
- 서버 복구 완료 메시지
- "시스템 복구 90% 완료. 마지막 인증키가 필요합니다. 위치: 롯데백화점 본점"

## 6. 방 3: 롯데백화점 본점

### 배경
- 이미지: room3_ext.jpg (롯데백화점 야경) → room3_lobby.jpg (로비)
- 분위기: 신비롭고 따뜻한 분위기로 전환
- 스토리: "단서를 따라 롯데백화점 본점에 도착했다. 로비에 들어서니 무언가 이상하다... 익숙한 느낌."

### 퍼즐: DX팀 최종 퀴즈
- 5문제 중 3문제 이상 맞추면 통과
- 팀 문화, 팀원 관련 퀴즈

**퀴즈 예시:**
1. "DX팀에서 매년 나트랑으로 여행 가는 사람은?" → 여승혁
2. "전세중 수석님의 특기는?" → 간식 먹기
3. "내몸혁명 콤비는 누구와 누구?" → 홍병우 & 박종택
4. "박예은 리더의 강아지 이름은?" → 재인
5. "올해 DX팀에 처음 합류한 사람은?" → 이정호
6. "문기준 리더가 좋아하는 브랜드는?" → 나이키
7. "김재환 책임이 매주 가는 곳은?" → 에버랜드

### 클리어 시
- 화면 전환 → 감동 엔딩으로 이동

## 7. 엔딩 (캐릭터별 분기)

### 공통 연출
1. 화면이 서서히 밝아짐
2. 팀원 이름이 하나씩 타이핑되듯 나타남
3. 공통 메시지: *"혼자 고생했다고 생각했지? 우린 항상 같이야."*
4. "— DX팀 일동" 표시

### 캐릭터별 개인 메시지

**홍병우 팀장 엔딩: "팀장의 무게"**
> "팀장님, 항상 앞에서 이끌어주셔서 감사합니다.
> 내몸혁명은 실패해도, 우리 팀은 항상 성공입니다.
> 오늘도 수고하셨어요, 팀장님."

**박종택 책임 엔딩: "끈기의 사나이"**
> "종택님, 포기하지 않는 그 끈기가 팀의 원동력입니다.
> 내몸혁명이든 장애 복구든, 끝까지 함께해줘서 고마워요.
> 올해 다이어트는... 내년에 다시 해요 ㅋㅋ"

**이상현 책임 엔딩: "말 한마디의 힘"**
> "상현님의 통화 한 통이 팀을 살렸습니다.
> 말이 많다고 했지만, 그 말 하나하나가 다 필요한 거였어요.
> 덕분에 아무도 혼자가 아니었습니다."

**전세웅 책임 엔딩: "더스틴 정복자"**
> "세웅님, 드디어 더스틴을 정복했습니다.
> 매일 고통받았지만 포기하지 않은 당신이 진짜 영웅이에요.
> 이제 빌드 에러 없는 평화로운 하루가 올 거예요... 아마도."

### 엔딩 후
- 클리어 시간 표시
- "다른 캐릭터로 다시 플레이" 버튼
- 팀원 크레딧 롤 (전체 13명 이름 + 직급)

## 8. UI/UX 디자인 가이드

### 컬러 팔레트
- 메인: #1a1a2e (다크 네이비) — 긴장감
- 서브: #e94560 (레드) — 경고/긴급
- 액센트: #ffd700 (골드) — 힌트/중요 정보
- 엔딩: #4ecdc4 (민트) + #ffffff — 따뜻함/희망

### 폰트
- 제목: 고딕 계열 굵은 폰트
- 본문: 가독성 좋은 기본 폰트
- 힌트/메모: 손글씨 느낌 (cursive fallback)

### 애니메이션
- 방 전환: 페이드 인/아웃
- 퍼즐 정답: 초록 플래시 + 체크마크
- 퍼즐 오답: 화면 흔들림 + 빨간 플래시
- 엔딩: 텍스트 타이핑 효과 + 이름 페이드인

### 사운드 (CSS 애니메이션으로 대체)
- 정전: 화면 깜빡임 효과
- 경고: 빨간 펄스 효과
- 정답: 밝은 하이라이트
- 엔딩: 별/빛 파티클 효과

## 9. 기술 구현 계획

### 파일 구조
```
├── index.html          # 단일 HTML (CSS + JS 내장)
├── images/
│   ├── room1.jpg       # 6 회의실
│   ├── room2.jpg       # 장교빌딩 서버실
│   ├── room3_lobby.jpg # 롯데 사옥 로비
│   ├── room3_ext.jpg   # 롯데백화점 본점 야경
│   ├── char_hong.jpg   # 홍병우 팀장
│   ├── char_park.jpg   # 박종택 책임
│   ├── char_lee.jpg    # 이상현 책임
│   └── char_jeon.jpg   # 전세웅 책임
├── README.md
└── DESIGN.md
```

### 핵심 구현 사항
- 순수 HTML + CSS + JavaScript (프레임워크 없음)
- 상태 관리: JS 객체로 게임 상태 관리
- 화면 전환: DOM 조작으로 화면 show/hide
- 이미지: `<img>` 태그로 images/ 폴더 참조 (파일 크기 최적화)
- 반응형: 모바일/데스크탑 대응
- 타이머: setInterval 기반 경과 시간 측정
