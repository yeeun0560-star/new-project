# 🚪 DX팀의 마지막 배포 — 방탈출 게임

## 프로젝트 개요

DX팀배 AI 콘테스트 출품작. Kiro를 활용한 바이브 코딩으로 제작한 웹 기반 방탈출 게임.

## 컨셉

> 금요일 밤, 긴급 장애 발생. 시스템이 하나씩 멈춰간다.
> 팀원들은 이미 퇴근했고, 당신 혼자 사무실에 남아있다.
> 하나씩 문제를 풀어가며 시스템을 복구해야 한다.
> 하지만 풀어갈수록 깨닫게 된다 —
> **사실 팀원들이 미리 남겨둔 힌트들이 곳곳에 있었다는 것을.**
> 아무도 당신을 혼자 두지 않았다.

## 게임 구성

### 캐릭터 선택 (4명)

| 캐릭터 | 직급 | 특성 | 엔딩 테마 |
|---|---|---|---|
| 홍병우 | 팀장 | "내몸혁명" 다이어트 리더 | 팀장의 무게 — 팀을 지킨 리더의 감동 |
| 박종택 | 책임 | "내몸혁명" 다이어트 파트너 | 끝까지 해내는 끈기 |
| 이상현 | 책임 | 통화왕, 말이 많음 | 소통으로 팀원을 하나로 모은 영웅 |
| 전세웅 | 책임 | 더스틴(Dustin)에 고통받는 중 | 고통의 끝에서 더스틴을 정복한 자 |

### 방 구성 (3개 스테이지)

| 방 | 실제 장소 | 테마 | 분위기 |
|---|---|---|---|
| 1 | 6 회의실 | 갑자기 불이 꺼진 회의실, 탈출 시작 | 불안 + 호기심 |
| 2 | 장교빌딩 서버실 | 장애 확산, 시간 압박 | 긴장 + 단서의 따뜻함 |
| 3 | 롯데백화점 본점 | 마지막 미션 + 감동 엔딩 | 감동 반전 |

### 감정 흐름

```
긴장감 ────────────────────────────────→ 감동
[방1: 불안/호기심] → [방2: 긴장/압박] → [방3: 반전/감동]
```

### 엔딩

마지막 문을 열면, 팀원들이 남긴 메시지:
> *"혼자 고생했다고 생각했지? 우린 항상 같이야. — DX팀 일동"*

캐릭터별 개인화된 엔딩 메시지 + 팀원 이름이 하나씩 등장하는 연출.

## DX팀 구성원

- **팀장**: 홍병우
- **수석**: 전세중 (간식 마스터)
- **책임**: 김재환, 이상현(통화왕), 전세웅(더스틴 고통), 박종택(내몸혁명)
- **리더**: 여승혁, 문기준, 정아름, 박예은, 이정호, 김하늘, 이성현

## 퍼즐 소재

- 팀원 특징 매칭
- 더스틴 에러 로그 해독
- 내몸혁명 관련 숫자 퍼즐
- 통화 기록 기반 힌트
- DX팀 문화 퀴즈

## 기술 스택

- 단일 HTML 파일 (CSS + JS 내장)
- 실제 장소/팀원 사진 포함 (images/ 폴더 참조)
- 반응형 디자인
- CSS 애니메이션 (엔딩 연출)

## 제작 배경

DX팀의 팀워크와 유대감을 게임으로 표현하고 싶었습니다. 긴장감 있는 방탈출 게임을 풀어가면서, 결국 "혼자가 아니었다"는 메시지를 전달합니다.
